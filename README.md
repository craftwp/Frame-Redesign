# Frame-Redesign
A front-end redesign project inspired by the Frame website, created as a portfolio showcase.

Built entirely with HTML and CSS, this project highlights clean code structure, semantic markup, and modern web design principles.

Getting Started

To run this project locally:

1. Clone the repository:

git clone https://github.com/craftwp/frame-redesign.git

2. Open the project folder.

3. Launch the "index.html" file in your browser.
